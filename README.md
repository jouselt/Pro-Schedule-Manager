# Pro Schedule Manager

A professional shift scheduling application built with vanilla JavaScript, HTML, and CSS. It utilizes Material Design components via CDN and several third-party libraries for functionalities like Excel/PDF export.

## Installation

This project does not require a build step. Simply open `index.html` in your web browser.

## Usage

- Import employee and shift data from CSV files.
- Manage schedules, employees, and shifts through dedicated tabs.
- Export schedules to CSV, Excel, image, or PDF formats.
